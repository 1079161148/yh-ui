YH-UI 核心组件库，基于 Vue 3.5+ 构建的高性能、可扩展的企业级 UI 解决方案。

---

## ✨ 特性

- **61+ 企业级组件**：覆盖绝大部分业务场景。
- **Composition API**：深度使用 Vue 3 组合式 API，逻辑清晰且极致轻量。
- **TypeScript**：全量类型定义，提供完美的开发提示。
- **Tree-shaking**：原生 ES Modules 支持，按需构建。

## 📦 安装

```bash
# pnpm
pnpm add @yh-ui/components
```
