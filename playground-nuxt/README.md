# Nuxt Playground for YH-UI

这是 YH-UI 的 Nuxt 3/4 测试环境。

## 运行

```bash
# 安装依赖
pnpm install

# 开发模式
pnpm dev

# 构建
pnpm build

# 预览构建结果
pnpm preview

# 生成静态站点
pnpm generate
```

## 测试内容

- ✅ 组件自动导入
- ✅ Composables 自动导入  
- ✅ 样式自动注入
- ✅ SSR 渲染
- ✅ Hydration 一致性
- ✅ Message/Notification 全局方法

## 访问

开发服务器: http://localhost:3000
