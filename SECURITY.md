# Security Policy

## Supported Release Line

YH-UI is currently maintained as a single active release line based on the latest published version.

Security fixes are prioritized for:

- the latest published package versions
- release candidates that are about to become the next stable line

Older versions may not receive backports.

## Reporting a Vulnerability

Please do not report security vulnerabilities in public GitHub issues.

Use one of these private channels instead:

1. Open a GitHub Security Advisory for this repository if that option is available to you.
2. Contact the repository maintainer privately through the GitHub account associated with this repository.

Please include:

- affected package name and version
- impact summary
- reproduction steps or proof of concept
- suggested mitigation if known

## What to Expect

We will aim to:

- acknowledge the report within 5 business days
- reproduce and assess severity
- prepare a fix or mitigation plan
- coordinate disclosure timing when a fix is ready

## Disclosure Policy

We prefer coordinated disclosure. After a fix is available, we will publish the relevant release notes and update users through the normal release channel.
