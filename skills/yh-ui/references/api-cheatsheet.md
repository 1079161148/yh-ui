## Theme

Use the real theme plugin and hooks:

```ts
import { ThemePlugin, useTheme } from '@yh-ui/theme'

app.use(ThemePlugin, {
  preset: 'blue',
  followSystem: true,
  persist: true
})
```

Common theme helpers include `useTheme`, `setThemePreset`, `toggleDarkMode`, `checkContrast`, and `getTextColorForBackground`.
