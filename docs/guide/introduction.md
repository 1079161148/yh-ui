# 简介

## 什么是 YH-UI？

YH-UI 是一个基于 Vue 3 的现代化组件库，融合了 市面组件库 的简洁性、Ant Design 的系统性、Naive UI 的现代性，致力于提供高性能、灵活定制的企业级组件解决方案。

## 特性

### 🚀 高性能
- 基于 Vue 3 Composition API 构建
- 支持完整的 Tree-shaking
- 按需加载，体积最小化
- 虚拟滚动支持大数据量场景

### 🎨 灵活定制
- 完善的 CSS 变量系统（`--yh-*` 前缀）
- 运行时主题切换
- 内置深色模式
- BEM 命名规范，样式隔离

### 📦 开箱即用
- 丰富的组件库
- API 设计简洁直观
- 完善的中文文档
- 丰富的示例代码

### 🔧 TypeScript 支持
- 完整的类型定义
- 智能的类型推导
- 友好的 IDE 提示

### 🌍 国际化
- 内置中英文语言包
- 支持自定义语言
- 运行时语言切换

## 设计理念

### 简洁性
API 设计遵循"约定大于配置"的原则，常用功能开箱即用，减少配置成本。

### 一致性
遵循统一的设计语言和交互规范，确保用户体验的一致性。

### 灵活性
提供丰富的定制能力，满足不同业务场景的需求。

### 高性能
从架构设计到实现细节，始终以性能为核心考量。
