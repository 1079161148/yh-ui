# AiActionGroup

`AiActionGroup` is a collection of action buttons for the bottom of AI reply bubbles, commonly used for message copying, regenerating, evaluation, etc., providing a compact and elegant layout.

<script setup lang="ts">
import { toJs, _T, _S } from '../../.vitepress/theme/utils/demo-utils'
import type { ActionItem } from '@yh-ui/components'

const handleAction = (key: string) => {
  console.log('Action Clicked:', key)
}

const customItems: ActionItem[] = [
  { key: 'share', icon: 'share', label: 'Share Chat' },
  { key: 'edit', icon: 'edit', label: 'Rewrite' },
  { key: 'delete', icon: 'delete', label: 'Delete Record', disabled: true }
]

const tsBasic = `<${_T}>
  <div style="background: var(--yh-bg-color-overlay); padding: 12px; border-radius: 8px;">
    <YhAiActionGroup
      :items="['copy', 'regenerate', 'thumb-up', 'thumb-down']"
      @click="handleAction"
    />
  </div>
</${_T}>

<${_S} setup lang="ts">
const handleAction = (key: string) => {
  console.log('Action Clicked:', key)
}
</${_S}>`

const jsBasic = toJs(tsBasic)

const tsVertical = `<${_T}>
  <div style="display: flex; gap: 20px;">
    <YhAiActionGroup
      direction="vertical"
      :items="customItems"
      @click="handleAction"
    />
  </div>
</${_T}>

<${_S} setup lang="ts">
const customItems = [
  { key: 'share', icon: 'share', label: 'Share Chat' },
  { key: 'edit', icon: 'edit', label: 'Rewrite' },
  { key: 'delete', icon: 'delete', label: 'Delete Record', disabled: true }
]
const handleAction = (key: string) => {
  console.log('Custom Clicked:', key)
}
</${_S}>`

const jsVertical = toJs(tsVertical)

const tsBubble = `<${_T}>
  <YhAiBubble content="This is an excellent response generated by AI!" direction="ltr">
    <template #footer>
      <YhAiActionGroup :items="['copy', 'thumb-up']" />
    </template>
  </YhAiBubble>
</${_T}>`

const jsBubble = toJs(tsBubble)
</script>

## Basic Usage

You can quickly generate action groups with preset styles through simple string arrays.

<DemoBlock title="Basic Usage" :ts-code="tsBasic" :js-code="jsBasic">
  <div style="background: var(--yh-bg-color-overlay); padding: 12px; border-radius: 8px;">
    <YhAiActionGroup
      :items="['copy', 'regenerate', 'thumb-up', 'thumb-down']"
      @click="handleAction"
    />
  </div>
</DemoBlock>

## Vertical Layout and Customization

Support custom icons, labels, and click logic.

Place the action group at the bottom of the conversation bubble.

## API

### Props

| Name        | Description       | Type                              | Default        |
| ----------- | ----------------- | --------------------------------- | -------------- |
| `items`     | Action items list | `ActionItem[]`                    | `[]`           |
| `size`      | Button size       | `'small' \| 'default' \| 'large'` | `'small'`      |
| `direction` | Layout direction  | `'horizontal' \| 'vertical'`      | `'horizontal'` |

### ActionItem Type

If it's a string, presets are supported: `'copy' | 'refresh' | 'regenerate' | 'share' | 'thumb-up' | 'thumb-down' | 'edit' | 'delete'`. If it's an object:

| Name       | Description        | Type      |
| ---------- | ------------------ | --------- |
| `key`      | Unique identifier  | `string`  |
| `icon`     | Icon name          | `string`  |
| `label`    | Text label         | `string`  |
| `disabled` | Whether disabled   | `boolean` |
| `tooltip`  | Hover tooltip text | `string`  |

### Events

| Event Name | Description                        | Callback Parameters               |
| ---------- | ---------------------------------- | --------------------------------- |
| `click`    | Triggered when a button is clicked | `(key: string, item: ActionItem)` |

### Slots

| Slot Name | Description                                 |
| --------- | ------------------------------------------- |
| `default` | Append custom content after the action list |

## Theme Variables

| Variable Name              | Default Value                |
| -------------------------- | ---------------------------- |
| `--yh-ai-action-gap`       | `8px`                        |
| `--yh-ai-action-icon-size` | `16px`                       |
| `--yh-ai-action-hover-bg`  | `var(--yh-fill-color-light)` |
